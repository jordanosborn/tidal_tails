# Computational Project
Part II Computational Project: Tidal Tails

## Required:

  1. C++ compiler (GNU g++)
  1. SDL2
  1. OpenGL
  1. cmake

## Instructions to build project:

```bash
	cd {project-directory}
	cmake .
	make
```
Project executable located in bin directory and is called main.
